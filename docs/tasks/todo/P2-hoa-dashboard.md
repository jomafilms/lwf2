## Problem

Dennis at Mountain Meadows HOA said: "An app we can recommend as approved for adoption by all associations." HOAs need to track which properties have complied, send reminders, and generate reports.

## Proposed Changes

### 1. HOA org type
- Create org with type "hoa"
- Define HOA boundary (draw on map or import from parcel data)
- Add properties within boundary

### 2. Compliance dashboard
- Grid/map view of all properties in the HOA
- Status per property: not started, plan submitted, approved, completed
- Compliance percentage: "47% of homes have Zone 0 plans"
- Trend over time (chart)

### 3. Communication tools
- Bulk email/notification: "Reminder: Zone 0 assessment due by June 1"
- QR code generation for mailer campaigns
- Link to register property in the system

### 4. Reporting
- Export compliance report (PDF/CSV)
- CCR-compatible format
- Grant documentation support
