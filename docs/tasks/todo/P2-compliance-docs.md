## Problem

Homeowners need proof of compliance for insurance. Landscapers need submittable documents for cities/HOAs. Current plan export is basic. Need formal compliance documentation.

## Proposed Changes

### 1. Wildfire Prepared Home checklist
- Based on IBHS/NFPA standards
- Zone 0-5 requirements, Zone 5-30, Zone 30-100
- Checkbox progress tracker
- Photo upload per section (before/after)

### 2. HOA submission package
- Formatted to match common HOA submission requirements
- Property address, zone map, plant list with fire scores
- Statement of compliance with relevant CCRs
- Landscaper signature block (if professional plan)

### 3. Insurance documentation
- "My property meets Zone 0 requirements because..."
- Auto-generated from plan data + Charisse's risk reduction text
- Character score explanation
- Suitable for insurance company review

### 4. City reporting format
- CWPP-aligned reporting
- Parcel ID + compliance status
- Aggregate-ready (feeds into P2-5 city analytics)
